# Web App (Next.js)

This folder contains the MVP frontend and server routes.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Environment

Use values from root `.env.example` in `web/.env.local`:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `IMMICH_BASE_URL`
- `IMMICH_API_KEY`
- `IMMICH_ADMIN_EMAIL`
- `IMMICH_ADMIN_PASSWORD`
