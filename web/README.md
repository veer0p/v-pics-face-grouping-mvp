# Web App (Next.js)

This folder contains the MVP frontend and server routes.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Environment

Use values from root `.env.example` in `web/.env.local`:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `R2_ENDPOINT`
- `R2_REGION`
- `R2_ACCESS_KEY_ID`
- `R2_SECRET_ACCESS_KEY`
- `R2_BUCKET_NAME`
